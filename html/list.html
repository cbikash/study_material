Introduction to HTML Lists

HTML lists allow you to group a set of related items together.

Unordered Lists

An unordered list is a collection of items where the order does not matter. It uses the <ul> tag.

  • Item 1
  • Item 2
  • Item 3

Ordered Lists

An ordered list is a collection of items where the order matters. It uses the <ol> tag.

  1. First item
  2. Second item
  3. Third item

Nested Lists

You can create lists inside lists. This is called nesting.

  • Item 1
    • Sub-item 1
    • Sub-item 2
  • Item 2

Conclusion

HTML lists are useful for organizing content. They come in unordered, ordered, and nested forms.
